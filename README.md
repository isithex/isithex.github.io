# isithex.github.io

Source code for my personal portfolio — [isithex.github.io](https://isithex.github.io)

Terminal-themed, bilingual (EN/AR), built from scratch with vanilla HTML/CSS/JS.

## Stack

`HTML` · `CSS` · `JavaScript` · `GitHub Pages`

## Structure
```
isithex.github.io/
├── index.html
├── style.css (or equivalent)

```

## Run locally
```bash
git clone https://github.com/isithex/isithex.github.io
cd isithex.github.io
open index.html in your browser
```
